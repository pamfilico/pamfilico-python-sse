# {{ cookiecutter.project_name }} — SSE Events

Server-Sent Events setup generated by `pamfilico-python-sse`.

## Usage

```python
from app.services.sse import emit_event
from app.services.sse.events import emit_record_updated, emit_table_updated

# In a route handler, after session.commit():
emit_event("updated_favorite_123", {"document_id": "123", "is_favorite": True})

# Or use the helpers:
emit_record_updated("favorite", document_id, is_favorite=True)
emit_table_updated("favorites")
```

## Environment

| Variable | Default | Description |
|----------|---------|-------------|
| `NEXTAUTH_URL` | `{{ cookiecutter.frontend_url }}` | Frontend base URL for SSE trigger |
